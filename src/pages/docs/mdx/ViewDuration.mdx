## 표시 시간

시간표의 기본 표시 시간은 **09:00 ~ 19:00**고 **한 칸은 10분**을 나타냅니다. 만약 예약이 표시 시간을 벗어난다면 화면에 나타나지 않습니다.

병원의 예약 시간이 기본 표시 시간과 다르면 **시간표 설정**에서 바꿀 수 있습니다.
